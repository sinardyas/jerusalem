Three layout directions for the operator (live control) window · macOS · from docs/MVP.md
Prototypes A / B / C are three layouts of the same screen — the operator (live control) window — with the same MVP elements (playlists/library, slide grid, live + next preview, video transport, panic keys). Pick one direction. The Slide Editor below is a separate screen, shown in the chosen Mac-native style. Everything is interactive: click a slide, press ← → / Space to navigate, B for black; in the editor, edit text on the canvas and try the inspector controls.
